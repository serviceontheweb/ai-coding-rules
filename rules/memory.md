# Rule: Memory

Use memory files carefully.

## LESSONS.md

Add durable lessons only. A lesson should prevent a future repeated mistake.

Do not add secrets, one-off notes, or uncertain guesses.

## CHANGELOG.md

Add completed user-visible work only.

Do not add an entry for experiments that did not change the project, unless the experiment documents a durable decision.

## Closeout

At the end of substantial work, check whether either file needs an update.
