# Roadmap

This roadmap keeps the project lightweight and governance-focused.

## Near Term

- Improve wiki navigation.
- Keep adapter guidance current.
- Add small policy examples as patterns stabilize.
- Add validation examples without requiring a runtime.

## Later

- Define optional schemas for policy files.
- Add link-checking guidance.
- Add more framework templates when they provide reusable value.

## Non-Goals

- Agent runtime implementation.
- Proprietary orchestration.
- Product-specific compliance systems.
- Large process manuals.
