# Claude Code Integration

Claude Code can use `CLAUDE.md` as a short bridge to `AGENTS.md`.

## Recommended Setup

Create a concise `CLAUDE.md`:

```md
# Claude Code Instructions

Follow the repository instructions in `AGENTS.md`.
```

Keep shared rules in `AGENTS.md`. Use Claude-specific files only for
tool-specific behavior.
