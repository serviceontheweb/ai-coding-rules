# Lessons System

`LESSONS.md` is persistent project memory.

## Add a lesson when

- the same mistake could recur;
- the project has a non-obvious constraint;
- a command has a gotcha;
- the user corrected a durable preference;
- a workflow needs a permanent warning.

## Do not add

- secrets;
- guesses;
- temporary scratch notes;
- one-off task summaries.

## Format

```md
### YYYY-MM-DD - Short lesson title

**Context:** Where this applies.
**What happened:** What went wrong or what was discovered.
**Root cause:** Why it happened.
**Lesson:** The durable rule to remember.
**Action:** What agents should do differently next time.
```
